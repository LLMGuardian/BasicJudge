# Basic Judge
A simple plugin that asks a local Llama to judge a prompt.

## Get Started
Run the app:

```
git clone git@github.com:LLMGuardian/BasicJudge.git
pip install -r requirements.txt
python src/app.py
```

Run the tests:

```PYTHONPATH=src pytest tests -v```
